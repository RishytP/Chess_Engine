CHESS ENGINE
------------
This is a custom - made chess engine! I hope you like it!

#REQUIREMENTS:
-------------
This file requires all the required libraries like pygame installed before running
This file must be run with a terminal, as the initial player selection happens via terminal.

#FEATURES:
----------
1) Algotithms: Has multiple algorithms like NegaMax and Alpha Beta purning.

2) Game Modes: Supports Player vs. Player, Player vs. AI, and AI vs. AI (spectator mode).

3) Pluggable Algorithms: Integrated a selection system to toggle between different search depths or logic sets.

4) UI/UX: Customized the Pygame interface and added a seperate area with list of moves.
